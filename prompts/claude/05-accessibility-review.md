# Review Accessibility Copy and Interaction

## Objective

Review learner instructions, focus order, timer language, reduced motion, contrast expectations, and error/status copy for accessibility barriers.

## Allowed Files

Read UI presenters/tests, English localization, `docs/02-LEARNER-EXPERIENCE.md`, and `docs/08-ACCESSIBILITY-LOCALIZATION.md`. Edit docs/copy only if authorized.

## Files Not To Change

Do not alter scenes, runtime behavior, art assets, project settings, or external repositories. Never inspect or modify the external AGROVATOR LMS repository.

## Required Output

Return a keyboard/screen-flow audit, plain-language copy changes, severity, and manual assistive-technology checks for humans.

## Required Tests

Check key existence and UI length constraints; require PlayMode/browser/manual evidence for behavior claims rather than inferring a pass from source.

## Definition of Done

Automated and human evidence are distinguished, no conformance certification is claimed, and all release blockers have owners.
