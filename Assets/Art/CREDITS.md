# Art credits and provenance

All four raster assets in this directory are original project artwork created on 14 July 2026 with OpenAI's built-in image generation tool. The selected outputs were locally cropped and downscaled for the Unity project. The Judge Aya sheet also received chroma-key background removal; UI sheets were normalized to deterministic equal-cell grids.

No third-party artwork, copyrighted character, commercial game interface, logo, watermark, or sprite derivative was used. The assets are licensed for use with this project under the OpenAI output terms applicable to the generating account. Project source code remains under the repository's own license.

The exact generation prompts, processing notes, dimensions, and project paths are recorded in `manifest.json` beside this file.
