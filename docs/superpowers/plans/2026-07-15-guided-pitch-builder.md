> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the active timed recognition exercise with an untimed Primary/Secondary guided builder that teaches learners to assemble, improve, present, and reflect on a four-part Smart School Garden pitch.

**Architecture:** Add a simulator-local, engine-independent `GuidedPitch` assembly for learner mode, validated authored content, draft/revision state, assessment, and phase rules. A new `GuidedPitchSessionController` composes that domain with the existing LMS bridge and payload types; thin UGUI presenters render one persistent guided-builder screen plus Results, while the current legacy dialogue classes remain untouched and inactive during this migration. The WebGL shell renders at CSS size times device pixel ratio, the garden is controlled to 16:9, and the UGUI layer switches to a stacked compact layout for constrained viewports.

**Tech Stack:** Unity `6000.5.3f1`, pure C# domain assemblies, UGUI, NUnit EditMode/PlayMode tests, deterministic editor scene generation, PowerShell test/build wrappers, Node.js source-contract tests, and Playwright Chrome/Edge smoke.
